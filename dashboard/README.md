# Starknet RPC Monitor Dashboard

Password-protected Next.js dashboard for monitoring Starknet RPC health and usage.

## Quick Start

```bash
cd dashboard
npm install
npm run dev
```

**Default Password**: `starknet2026` (change in `.env.local`)

Open http://localhost:3000

## Features

✅ Password protected access
✅ Real-time health monitoring
✅ Usage statistics with charts
✅ Auto-refresh (10s/30s/1m/5m)
✅ Responsive design

## Production

```bash
npm run build
npm start
```

See full documentation in dashboard folder.
